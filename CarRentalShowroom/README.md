# Car Rental Management System

This project is developed as an academic application to demonstrate the use of
Data Structures with a Graphical User Interface (GUI).

---

## Project Information

**Project Name:** Car Rental Management System  
**Field / Major:** Cybersecurity  
**Group:** 4

**Description:**  
The project is a **Car Rental System** that allows managing rental cars using
a graphical user interface.  
It applies different **Data Structures** to organize and manage data efficiently.

---

## Data Structures Used

- Array
- Linked List
    - Singly Linked List
    - Doubly Linked List

---

## Project Structure

All classes are organized under the following path:
src/main/java/app/ds/model/repo/ui/util



This structure helps in maintaining clean code separation between:
- Models
- Data Structures
- Repositories
- User Interface
- Utilities
